


分类：税务稽查工作规程


// 第一节   税务稽查案源管理


题型：判断题

1.随机抽样有两个显著特点：一是遵守机会均等原则，二是可以从数量上推断总体。( √ )

2.发票案件的查处由案发地的税务机关负责。( √ )

3.检举内容不详，无明确线索或者内容重复的，暂存待办。( √ )

4.选案部门应当建立案源信息档案，对所获取的案源信息实行分级管理。( × )


// 第二节   税务稽查实施


题型：判断题

1.税务机关若要将纳税人当年度的账簿资料调回税务机关检查的，应在3个月内退还。( × )

2.在填写《调取账簿资料清单》时，调取账簿资料的数量应以阿拉伯数字填写。( × )

3.税务机关对从事生产、经营的纳税人的纳税情况依法进行税务检查时，发现纳税人有逃避纳税义务行为，并有明显的转移、隐匿其应纳税的商品、货物以及其他财产或者应纳税的收入的迹象的，可以采取税收保全措施。( × )

4.对被检查对象要求检查人员回避的，检查人员是否回避，由上一级税务机关局长审定。( × )

5.检查完毕，检查部门应当将《税务稽查报告》、《税务稽查工作底稿》及相关证据材料，在3个工作日内移交审理部门审理，并办理交接手续。( × )

6.被查对象拒不打开或者拒不提供的，经稽查局局长批准，可以采用适当的技术手段对该电子信息系统进行直接检查。( √ )

7.调查取证时，不得违反法定程序收集证据材料；不得以偷拍、偷录、窃听等手段获取侵害他人合法权益的证据材料；不得以利诱、欺诈、胁迫、暴力等不正当手段获取证据材料。( √ )

8.调取账簿、记账凭证、报表和其他有关资料时，应当向被查对象出具《调取账簿资料通知书》，并填写《调取证据专用收据》交其核对后签章确认。( × )

9.提取证据材料复制件的，应当由原件保存单位或者个人在复制件上注明“此件由我公司提供，经核无误”，并由提供人签章。( × )

10.询问应当由两名以上检查人员实施并应当向被询问人送达《询问通知书》。( × )

11.笔录可使用能够长期保持字迹的书写工具书写，也可使用计算机记录并打印，陈述或者证言应当由陈述人或者证人逐页签章、捺指印。( × )

12.调取视听资料时，应当调取有关资料的原始载体，不得调取复制件。( × )

13.以电子数据的内容证明案件事实的，应当要求当事人将电子数据打印成纸质资料，在纸质资料上注明数据出处、打印场所，注明“与电子数据核对无误”，并由当事人签章。( √ )

14.制作现场笔录、勘验笔录，应当载明时间、地点和事件等内容，并由检查人员签名或当事人签章。( × )

15.检查从事生产、经营的纳税人以前纳税期的纳税情况时，发现纳税人有逃避纳税义务行为，并有转移、隐匿其应纳税的商品、货物以及其他财产或者应纳税收入迹象的，经所属税务局局长批准，可以依法采取税收保全措施。( × )

16.税务检查部门可以填制《税收违法案件终止检查审批表》，附相关证据材料，直接经稽查局局长批准后，终止检查。( × )


// 第三节   税务稽查审理


题型：判断题

1.税务行政处罚事项告知和听证程序均是税务行政处罚的法定必经程序。( × )

2.税务机关立案查处的各类税务违法案件都必须经过稽查审理环节。( √ )

3.在具体实践中，对税务稽查涉案税额达到一定标准的案件及疑难案件，由审理部门提请稽查局长审定。( × )

4.本机关工作人员查阅税务稽查档案应当征得档案管理部门负责人同意。( × )

5.税务稽查案卷应当在立卷次年6月30日前移交所属税务局档案管理部门保管。( √ )

6.税务稽查工作规程所称以上、日内，不包括本数。( × )


// 第四节   税务稽查执行


题型：判断题

1.留置送达与直接送达具有相同的法律效力。( √ )

2.采取委托送达的，受托机关责任人在送达回证上签字备注的时间即为送达的日期。( × )

3.对稽查局查补税款和滞纳金也可以申清人民法院强制执行。( × )

4.税务机关可以以纳税人的名义向人民法院申请对第三人行使代位权。( × )

5.边防检查站阻止欠税人出境的期限为10天。( × )

6.执行部门接到审理部门移送的税务处理文书后，应对文书的完整性等进行审核，对不符合执行受理条件的案件，应及时将处理文书退回审理部门。( × )

7.稽查执行由各级稽查局专门负责案件实施的部门组织实施。( × )

8.填写《税务文书送达回证》时，受送达人为单位的，“受送达人”一栏，填写单位简称。( × )

9.对税务稽查认定当事人多申报缴纳税款的，被执行人应按照税务处理决定规定的期限和方式，到实施稽查的稽查局办理退库手续，或将多缴的税款抵缴应纳税款，并将税款退库凭证的复印件或抵缴税款的复印件连同税务稽查调整账务记录复印件报送稽查执行部门。( × )

10.填制《税务文书送达回证》是稽查局送达税务稽查文书时必须履行的法定手续。( √ )

11.留置送达方式中，见证人可以是非本案执行人员的税务机关工作人员。( × )

12.稽查局在采取税收强制执行措施过程中，如滥用职权违法采取税收强制执行施，或者采取税收强制执行措施不当，使纳税人、扣缴义务人或者纳税担保人的合法权益遭受损失的，税务机关应依法对其直接损失和间接损失承担赔偿责任。( × )

13.对到外县（市）从事生产、经营而未向营业地税务机关报验登记的纳税人，由税务机关核定其应纳税额，责令缴纳；不缴纳的，税务机关可以扣押其价值相当于应纳税款的商品、货物。扣押后仍不缴纳应纳税款的，经县以上税务局(分局)局长批准，依法拍卖或者变卖所扣押的商品、货物，以拍卖或者变卖所得抵缴税款。( √ )

14.当事人对税务行政处罚决定逾期不予履行的，作出税务行政处罚决定的税务机关可以申请人民法院强制执行。( × )

15.如果当事人既有银行存款又有商品、货物或者其他财产的，一般情况下，可先采取通知银行扣缴税款的措施；如果银行存款不足以扣缴的，则可将两项措施并用。( √ )

16.欠缴税款的纳税人因怠于行使到期债权，对国家税收造成损害的，税务机关可以依照《合同法》的规定直接自行行使代位权。( × )

17.阻止欠税人出境由县级以上（不含县级）税务机关填写《阻止欠税人出境布控申请表》，报省、自治区、直辖市税务机关审核批准即可。( × )

18.税务机关在行使收缴或停止供应发票权时，必须以从事生产、经营的纳税人、扣缴义务人有《税收征管法》规定的涉税违法行为且拒不接受税务机关处理为前提。( √ )

19.在税务行政强制执行过程中，不发生执行和解问题。( √ )

20.实施扣押、查封、拍卖或变卖等税收强制执行措施时，应当通知被执行人或者他的成年家属到场，否则，不能直接采取扣押、查封措施,但被执行人或家属接到通知后拒不到场的,不影响执行。( √ )

21.《税务文书送达回证》“受送达人签名或盖章”一栏，受送达人为单位的，必须由法定代表人签名盖章。( × )

22.纳税人、扣缴义务人、纳税担保人未按规定期限缴纳税款的，由税务机关发出限期缴纳税款通知书，责令限期缴纳税款，但最长期限为7日。( × )

23.被查封的财产，税务机关不允许被执行人继续使用。( × )

24.被执行人确实没有财产抵缴税款或者依照破产清算程序确实无法清缴税款，或者有其他法定终结执行情形的，稽查局可以填制《税收违法案件终结执行审批表》，依照国家税务总局规定权限和程序，经税务局相关部门审核并报所属税务局局长批准后，终结执行。( √ )

25.中止执行情形消失后，应当及时填制《税收违法案件解除中止执行审批表》，经稽查局局长批准后，恢复执行。( √ )

26.税务稽查执行是税务稽查执行环节根据审理环节作出的稽查案件处理意见，将税务处理决定文书送达被执行人，并督促其依法履行税务处理决定的活动。( √ )

27.出现拒绝签收情形的， 《税务文书送达回证》“受送达人拒收理由和日期”一栏，由受送达人填写此栏。( × )

28.税务处理决定文书一经审定，即具有法律效力。( × )

29.纳税人超过应纳税额缴纳的税款，税务机关发现后应当在1个月内退还。( × )

30.税务机关解除查封、扣押措施时，税务人员应与被查封、扣押商品、货物或其他财产的纳税人当场清点查封、扣押的物品。( √ )

31.采取强制执行措施，应当履行审批程序报经县以上税务局（ 分局）局长批准。( √ )

32.税务稽查强制执行的范围不但包括税款、滞纳金，对罚款也可以强制执行。( √ )

33.纳税人采取偷税、抗税、骗税手段的，税务机关可以无限期追征其未缴或者少缴的税款、滞纳金或者所骗取的税款。( √ )

34.税务机关采取税收保全措施和强制执行措施必须依照法定权限和法定程序，不得查封、扣押纳税人个人及其所扶养家属的住房和用品。( × )

35.2010年6月15日，某法人代表王某要求到美国考察，税务机关发现其还有欠税，税务机关可以要求其提供纳税担保，如不提供担保，又不缴清欠税的，可以通知出境管理机关阻止其出境。( √ )

36.纳税人欠缴的税款发生在纳税人以其财产设定抵押、质押或者纳税人的财产被留置之后的，税收应当先于抵押权、质权、留置权执行。( × )

37.税务机关依照征管法第53条第一款规定行使代位权、撤销权的，可以免除欠缴税款的纳税人尚未履行的纳税义务和应承担的法律责任。( × )

38.对查封的商品、货物或者其他财产，税务机关可以指令被执行人负责保管，保管责任由税务机关承担。( × )

39.税收违法行为轻微，对当事人不予税务行政处罚的，制作并送达《不予税务行政处罚决定书》。( √ )

40.稽查局拍卖、变卖被执行人商品、货物或者其他财产前，应当拟制《拍卖/变卖抵税财物决定书》，经所属税务局局长批准后送达被执行人，予以拍卖或者变卖。( √ )
