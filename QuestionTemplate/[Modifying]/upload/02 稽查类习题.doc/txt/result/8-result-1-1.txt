


分类：主要行业与特殊业务的税务稽查


// 第一节   房地产行业


题型：判断题

1.某房地产开发企业2010年当年已结转收入8000万，当年还有销售未完工产品取得预收帐款2000万,该企业2010年可以扣除的广告费为1500万.( √ )

2.2011年10月9日，甲公司与张先生签订购房合同，合同总价款120万元，合同约定： 张先生应在签约的3日内支付合同价款的40%，其余款项分别在11月份12月份各按30%支付。张先生10月、11月均按合同约定支付了购房款，到了12月，张先生因资金困难，暂时无法支付剩余的款项，甲公司按实际收到的价款84万元记作销售收入。( × )

3.某房地产公司2010年8月投入A项目开发，于2011年8月竣工, 从2011年10月项目开发起即采用按揭贷款方式对外销售，2011年10月收到首付款500万元计入“预收账款”， 2011年12月收到银行划转按揭贷款2000万元，计入“长期借款”。 ( × )

5.某房地产企业将成本为100，市场价为150的商品房用于本企业的固定资产，该固定资产计提折旧的计税基础只能是100.( √ )

6.企业销售未完工开发产品取得的收入按按预计计税毛利率分季（或月）计算出预计毛利额征税属于预征企业所得税,待实际结转收入时可以多退少补. ( × )

7.房地产开发企业一次性取得了3年租金90万，该企业可以选择在第一年确认90的收入，也可以选择确认30万的收入。( √ )

8.房地产企业发生的园林绿化费用，尚未取得全额发票，允许作为预提费用税前扣除. ( × )

10.土地增值税清算时，已经计入房地产开发成本的利息支出，应调整至财务费用中计算扣除。 ( √ )


// 第二节  建筑安装业的检查方法


题型：判断题

1.签订虚假合同.协议，虚列、多列或重复列支成本费用的主要检查方法之一是采取工程成本总量控制法，调查了解当地同类开发产品的平均建筑成本，对价格过高的建筑成本进行重点审查。( √ )

2.发包人工程竣工结算向承包人付款时，按合同约定的期限（一年）扣留质量保证金，待工程交付使用质保期到期后清算。由于质保期内如有返修，发生费用应在质保金内扣除。因此，承包人的质保金在竣工结算时，不应缴营业税，应在质保期到期后按清算金额计缴营业税。( √ )

3.总包人将建筑工程分包给其他单位的，其分包工程的营业税由应总包人扣缴。( × )

4.纳税人将建筑工程分包给其他单位的，以其取得的全部价款和价外费用扣除其支付给其他单位的分包款后的余额为营业额。( √ )

5.纳税人提供的建筑业劳务，应当向其机构所在地的主管税务机关申报缴纳营业税。( × )

6.纳税人提供建筑业劳务，采取预收款方式的，其纳税义务发生时间为该预收款转为工程结算收入的当天。( × )

7.纳税人提供建筑业劳务，自应当申报纳税之月起超过3个月没有向应税劳务发生地的主管税务机关申报纳税的，由其机构所在地或者居住地的主管税务机关补征税款。( × )

8.总包人2008年、2009年分别将承包的工程分包给其他单位的，该总包人对这两年度的分包工程应纳的营业税均负有扣缴义务。( × )

9.建安企业主营业务收入核算的会计科目有：“主营业务收入”、“产品销售收入”和“工程结算收入”。 ( × )

10.建安企业的财务核算，规模大的建筑安装企业一般实行公司、工区 (分公司、工程处)、施工队(项目经理部)三级核算体制。( √ )


// 第三节   资本交易检查方法


题型：判断题

1.企业重组，是指企业在日常经营活动以外发生的法律结构或经济结构重大改变的交易，包括企业法律形式改变、债务重组、股权收购、资产收购、资产置换、合并、分立等。( × )

2.企业重组的税务处理区分不同条件分别适用一般性税务处理规定和特殊性税务处理规定。( √ )

3.企业发生其他法律形式简单改变的，可直接变更税务登记，除另有规定外，有关企业所得税纳税事项（包括亏损结转、税收优惠等权益和义务）由变更后企业承继，但因住所发生变化而不符合税收优惠条件的除外。( √ )

4.企业分立相关企业的亏损可以相互结转弥补。( × )

5.同一控制下且不需要支付对价的企业合并，由于不能满足企业股东在该企业合并发生时取得的股权支付金额不低于其交易支付总额的85%的条件，因此不能选择特殊性税务处理。( × )

6.企业分立，被分立企业所有股东按原持股比例取得分立企业的股权可以选择特殊性税务处理。( √ )

7.企业发生符合本通知规定的特殊性重组条件并选择特殊性税务处理的，当事各方应在该重组业务完成当年企业所得税年度申报时，向主管税务机关提交书面备案资料，证明其符合各类特殊性重组规定的条件。企业未按规定书面备案的，一律不得按特殊重组业务进行税务处理。( √ )

8.企业在重组发生前后连续12个月内分步对其资产、股权进行交易，应根据实质重于形式原则将上述交易作为一项企业重组交易进行处理。( √ )

9.可由合并企业弥补的被合并企业亏损的限额=被合并企业净资产公允价值×截至合并业务发生当年年末国家发行的30年期限的国债利率。( × )

10.重组交易各方按对交易中股权支付暂不确认有关资产的转让所得或损失的，其非股权支付仍应在交易当期确认相应的资产转让所得或损失，但是，不用调整相应资产的计税基础。( × )
