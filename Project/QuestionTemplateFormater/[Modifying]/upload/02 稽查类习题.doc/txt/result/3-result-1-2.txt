


分类：税务稽查证据


题型：判断题

1.合法性是指证据的收集主体和证据形式合法。( × )

2.证明税收违法行为的凭证、账簿、报表、发票、合同、文件是物证。( × )

3.任何猜测.幻想和虚构，都不是客观事实，均不能作为合法证据使用。( √ )

4.常见的当事人陈述有证人的询问笔录、证人提供的口头或书面证词等。( × )

5.现场提取的发票当场鉴定后，出具的鉴定结论属于证据中的现场笔录。( × )

6.稽查人员现场制作的《实物盘点表》，属于证据中的勘验笔录.现场笔录。( √ )

7.需要以有形载体形式固定电子数据的，应当与提供电子数据的个人、单位的法定代表人或者财务负责人一起将电子数据复制到存储介质上并封存，同时在封存包装物上注明制作方法、制作时间、制作人、文件格式及长度等，注明“与原始载体记载的电子数据核对无误”，并由电子数据提供人签章。( √ )

8.税务稽查工作中，物证是最重要,数量最多的证据。( × )

9.按照证据与案件主要事实之间的关系，可将其分为原始证据和传来证据。( × )

10.凡是能够直接证明案件主要事实的证据，属于直接证据。( √ )

11.税收违法案件中的视听资料属于言词证据。( × )

12.按照证据的表现形式，可以将其分为直接证据和原始证据。( × )

13.其他证人证言优于与当事人有亲属关系或者其他密切关系的证人提供的对该当事人有利的证言。( √ )

14.根据证据效力的规定，复制件、复制品效力与原件相同。( × )

15.鉴定结论、现场笔录、勘验笔录、档案材料以及经过公证或者登记的书证优于其他书证.视听资料和证人证言。( √ )

16.法定鉴定部门的鉴定结论与其他鉴定部门的鉴定结论有同等效力。( × )

17.法庭主持勘验所制作的勘验笔录优于其他部门主持勘验所制作的勘验笔录。( √ )

18.不能正确表达意志的证人提供的证言不能单独作为定案依据的证据。( × )

19.凡采用偷拍、偷录等手段获取的证据材料都不能单独作为定案的依据。( × )

20.视听资料主要包括录音资料、录像资料、电子计算机储存资料.运用专门技术设备得到的信息资料等。( √ )

21.出庭作证的证人证言优于未出庭作证的证人证言。( √ )

22.国家机关以及其他职能部门依职权制作的公文文书优于其他书证。( √ )

23.收集物证时，可不收集原物，只要收集与原物核对无误的复制件或证明该物证的照片、录像等其他具有证明效力的证据。( × )

24.难以识别是否经过修改的视听资料不能作为定案依据的证据。( × )

25.无法与原件、原物核对的复制件或者复制品不能作为定案的依据。( × )

26.原始证据优于传来证据。( √ )

27.生效的人民法院裁判文书或者仲裁机构裁决文书确认的事实，双方事人没有异议后的，才可以作为定案依据。( × )

28.收集的报表、会计账簿、图纸、专业技术资料、科技文献等书证，应当附有说明出处等相关信息的材料并盖章和押印。( √ )

29.经税务机关通知申报而拒不申报造成不缴或者少缴应纳税款的行为属于偷税行为。( √ )

30.伪造发票的复印件不得作为出庭证据。( × )

31.偷拍的影像只要不侵害他人合法权益可以作为证据。( √ )

32.伪造的发票属于书证的范围。( × )

33.现场笔录无当事人签名不得作为证据。( × )

34.需要取得境外资料的，稽查局可以提请国际税收管理部门依照税收协定情报交换程序获取，或者通过我国驻外机构收集有关信息。( √ )

35.勘验笔录必须由稽查人员当场制作，不得由他人代为制作，特殊情况可事后补作。( × )

36.抗税行为主体只能是个体户。( × )

37.自然规律及定理可以直接作为认定事实的证据( √ )

38.对行政诉讼采用优势证明标准的证明标准( × )

39.法律规定某些证据不能单独作为认定案件事实的依据，只有在其他证据与其印证时，才能认定案件事实，这是证据的非法证据排除规则。( × )

40.根据录像资料收集与固定的要求,对取得的视听资料，应保持事实的完整性、真实性，可以进行剪辑、拼接等。( × )
